# Build from Source Code

To build this project. 

First clone this repo.

Build the code like a regular maven project.

```bash
$ mvn clean package
```

Then execute the last jar

```bash
$ java -jar target/parquetfx-0.1.0-SNAPSHOT-jar-with-dependencies.jar
```
